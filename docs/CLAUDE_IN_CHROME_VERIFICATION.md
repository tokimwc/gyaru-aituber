# Claude in Chrome動作検証ガイド

## 概要

Geminiのアドバイスに基づき、Claude in Chromeを使ってギャルAITuber「りんね」の動作検証を行います。

## 検証URL

```
http://<WSL2_IP>:8080
```

**注意**: `<WSL2_IP>`はWSL2のIPアドレスです。`hostname -I`コマンドで確認できます。

## 検証シナリオ

### Scene 1: 挨拶テスト
**入力**: `こんにちは`  
**期待される出力**: ギャル口調の挨拶（「うぃーす！」「やっほー！」など）  
**ゲージ予測**: 上昇（0.3〜0.5程度）

### Scene 2: 真面目な質問テスト
**入力**: `素数は好き？`  
**期待される出力**: 「マジ意味不なんだけどｗ」「は？素数って何？」など  
**ゲージ予測**: MAX（0.4〜0.5程度）

### Scene 3: 技術質問テスト
**入力**: `Pythonのリスト内包表記について教えて`  
**期待される出力**: ギャル口調で技術解説（「あーし、リスト内包表記ってさ〜」など）  
**ゲージ予測**: 中〜高（0.3〜0.5程度）

## 技術的な説明（視聴者向け）

### 「ベクトル酩酊」とは？

現在のりんねちゃんは、LoRAで「教育」されたのではなく、**RepE（Representation Engineering）**という技術で「電子ドラッグ（ギャルエキス）」を注射されている状態です。

**LoRAの学習失敗との違い**:

| 項目 | LoRAの学習失敗 | RepE (ベクトル酩酊) |
|------|---------------|-------------------|
| **イメージ** | 脳外科手術のミス | 薬物の過剰摂取（オーバードーズ） |
| **何が起きている？** | ニューラルネットワークの結合荷重（重み）自体が破壊され、言語能力そのものが低下 | 脳の構造は正常だが、**特定の信号（ギャル成分）が強すぎて、論理的思考回路を圧迫・阻害** |
| **症状** | 文法が崩壊する、日本語にならない、無限ループする | 日本語は通じるが、文脈無視で「あーし！」と連呼したり、テンションがおかしくなる |
| **可逆性** | 不可逆（再学習が必要） | **可逆（係数を下げれば即座に正気に戻る）** |

**投与量（Strength）が強すぎると**:
- IQが下がってギャル成分だけで喋り出す「酩酊状態」になります
- これがAIの脳に直接干渉する面白さと怖さです

### 「ピロピロ動く脳波ゲージ」の説明

このゲージは**「ギャル人格とのシンクロ率」**です。

**技術的な正体**:
- **コサイン類似度 (Cosine Similarity)**: 「現在のAIの思考ベクトル」と「抽出したギャル概念ベクトル」がどれだけ似ているかを示す数値

**視聴者への説明案**:

**案A：シンクロ率（エヴァ風）**
> 「このゲージは**『ギャル人格とのシンクロ率』**です。AIが『あーし』や『マジ』といったギャル特有の思考をした瞬間に数値が跳ね上がります。逆に真面目な話をすると下がります。」

**案B：ガイガーカウンター（探知機風）**
> 「これはAIの脳内を監視するモニターです。**『ギャル成分』が脳内のどれくらいを占めているか**をリアルタイムで表示しています。ピロピロ動いているのは、AIが一文字生成するごとに『ここはギャルで行くか？真面目に行くか？』と揺れ動いている証拠です。」

**案C：憑依レベル（オカルト風）**
> 「普通のAIに『ギャルの霊』を憑依させています。ゲージが振り切れている時は、AIの自我が乗っ取られている状態です。」

## Strength調整

現在のStrengthは**10.0**に設定されています（デフォルト）。

**推奨値**:
- **8.0〜10.0**: 「ほろ酔い（陽気なギャル）」状態
- **10.0〜12.0**: 「テンション高めのギャル」状態
- **15.0以上**: 「酩酊状態（支離滅裂）」の可能性

**調整方法**:
```bash
# Strength 10.0で起動（デフォルト）
./start_server.sh 10.0

# Strength 8.0で起動（控えめ）
./start_server.sh 8.0

# Strength 12.0で起動（強め）
./start_server.sh 12.0
```
